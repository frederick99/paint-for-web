Paint4Web v1.0

README

===========================================================

Summary:

    Paint4Web is a web application for simple image editing,
    allowing users to be creative using the limited yet
    versatile set of tools. It does this without the hassle
    of installing any software locally since it can run on
    a web browser natively.

===========================================================

Installation steps:

    No kind of local installation is required apart from a
    web browser.

===========================================================

How to run:

    Go to https://frederick99orig.github.io/paint-for-web/

===========================================================
